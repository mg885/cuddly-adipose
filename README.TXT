TOOLBOX FOR TFYY51, 2015
------------------------
This archive contains tools for the 2015 TFYY51 course. Look at the test_create_synth_images.m example to start with.
Many useful operations are already included in this toolbox, so check here first to see if something you want is already implemented.
To get help for a function, type >> help <function name> (or look in the source code)

For further information, contact marcus.wallenberg@liu.se
